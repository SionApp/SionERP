# SionERP

## Levantar el proyecto

### Frontend

Ir a la raíz del proyecto y ejecutar:

```bash
yarn dev
```

### Backend (API)

Ir a la carpeta del backend y ejecutar:

```bash
cd apps/backend-go
go run .
```

> **Nota:** Para información sobre Base de Datos y Migraciones, ver el [README del backend](./apps/backend-go/README.md).
