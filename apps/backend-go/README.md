# Backend (Go)

Este es el backend del proyecto SionERP, escrito en Go.

## Ejecución

Para levantar el API, simplemente corre:

```bash
go run .
```

## Base de Datos (Supabase)

Recuerda que debes tener levantado Supabase y correr las migraciones si es necesario.

### Migraciones

Las migraciones se encuentran en `supabase/migrations`.
Asegúrate de aplicarlas para tener la base de datos actualizada.
